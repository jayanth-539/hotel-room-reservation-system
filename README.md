# Hotel Room Reservation System

## Features
- Book up to 5 rooms
- Same floor priority
- Minimum travel time logic
- Cross-floor allocation
- Random occupancy generation
- Reset booking
- SQLite database

## API Endpoints
GET /rooms  
POST /book  
POST /random-occupancy  
POST /reset  

## Run Locally
pip install -r requirements.txt  
python app.py
